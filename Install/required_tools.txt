Required applications and modules:

1. Applications:

java 1.7 (check java version: java -version)

bwa: https://sourceforge.net/projects/bio-bwa/files/bwa-0.7.15.tar.bz2/download 
samtools: https://sourceforge.net/projects/samtools/files/samtools/1.3.1/samtools-1.3.1.tar.bz2/download 
bedtools2: https://github.com/arq5x/bedtools2/releases/download/v2.25.0/bedtools-2.25.0.tar.gz
prinseq: https://sourceforge.net/projects/prinseq/files/standalone/prinseq-lite-0.20.4.tar.gz/download
ABRA: https://github.com/mozack/abra/releases/download/v0.97/abra-0.97-SNAPSHOT-jar-with-dependencies.jar 
R: https://cran.r-project.org/src/base/R-3/R-3.2.1.tar.gz
R packages: ggplot2, naturalsort

Samtools version must be 1.0 or above. The version below 1.0 uses a different
command syntax and will not work in the pipeline.

Except java and R packages, the installation of the above tools is included in script install_app.sh as reference. 
Downloading of some files (bedtools and ABRA) could fail in the AWS cloud.

2. Perl and Python modules: 

To check the install status of Perl modules, run check_perlmod.pl. 
To install required Perl modules, pysamstats, and Python modules, run install_mod.sh as root.
