These are the tools that must be present or installed:

1. Applications:

java 1.7 (check java version: java -version)

bwa: https://sourceforge.net/projects/bio-bwa/files/bwa-0.7.15.tar.bz2/download 
samtools: https://sourceforge.net/projects/samtools/files/samtools/1.3.1/samtools-1.3.1.tar.bz2/download 
bedtools2: https://github.com/arq5x/bedtools2/releases/download/v2.25.0/bedtools-2.25.0.tar.gz
prinseq: https://sourceforge.net/projects/prinseq/files/standalone/prinseq-lite-0.20.4.tar.gz/download
ABRA: https://github.com/mozack/abra/releases/download/v0.97/abra-0.97-SNAPSHOT-jar-with-dependencies.jar 
picard 1.1197 to work with java 1.7:
	https://sourceforge.net/projects/picard/files/picard-tools/1.119/picard-tools-1.119.zip/download

pysamstats (see install_mod.sh)

R: https://cran.r-project.org/src/base/R-3/R-3.1.0.tar.gz
R packages: ggplot2, reshape2, naturalsort

Except java, pysamstats and R packages, the installation of the above tools is included in script
install_app.sh as reference. Some apps may fail to install.

2. Perl and python: 

To check the status of perl modules, run check_perlmod.pl. 
To install required perl modules, run install_mod.sh as root.
