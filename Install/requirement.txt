These are the tools that must be present or installed:

1. Applications:

java 1.7
bwa
samtools
bedtools2
prinseq
ABRA: abra-0.94-SNAPSHOT-jar-with-dependencies.jar
picard 1.107 or other to work with java 1.7
pysamstats (see install_mod.sh)

2. Perl and python: 

To check the status of perl modules, run check_perlmod.pl. 
To install required perl modules, run install_mod.sh as root.

3. R and packages:

ggplot2
reshape2
naturalsort
